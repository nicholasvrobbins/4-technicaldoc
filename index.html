Introduction

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS Example: .hi {font-size: 32px}

HTML & CSS

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

CSS is among the core languages of the open web and is standardized across Web browsers according to W3C specifications. Previously, development of various parts of CSS specification was done synchronously, which allowed versioning of the latest recommendations. You might have heard about CSS1, CSS2.1, CSS3. However, CSS4 has never become an official version.

HTML Basics

HTML uses "markup" to annotate text, images, and other content for structurizing what we see in a web browser. HTML markup includes special "elements" such as head, title, body, header, footer, article, div, img, and many, many more.

CSS Basics

CSS uses HTML elements to "attach to" and then stylizes them by changing their color, shape, sizing, padding, margin and even animations! With the two of them together, we can create some beautiful, animated and well structured websites, albeit a bit lacking on user interactivity (we'll get to more on that in the JavaScript section). Some examples of CSS in action look like:

.body {background-color: lightgray} .container {display: flex} .btn {margin: 1rem}

Applied Visual Design

Taking the time to learn proper visual design is absolutely essential. With it you'll know how to combine typography, color theory, graphics, animations, page layout and more to help bring your website's message to life.

CSS Flexbox

CSS Flexbox is a powerful, well-supported layout method that was introduced with the latest version of CSS, CSS3. It makes it easier than ever to center elements on the page and create a dynamic user interface that shrinks and expands automatically.

CSS Frameworks

After you've spent some time learning the basics of HTML & CSS, you've learned Flexbox it's time to move onto a CSS Framework. With so many to choose from it can be daunting at first, but I'd recommend Bootstrap for beginners as that's the most popular framework and is quite user friendly. Think of frameworks as a toolbox of various items. The way a construction worker might not use every tool in his toolkit for one specific job, but might need it later for a different job is how I think about using these frameworks. Also I do keep in mind that while things like changing colors to be brand specific are great, many developers much more experienced than myself have figured out the "optimal" line-height, padding sizes, font-sizes, etc., and it's best to learn these universal truths before bending them as you see fit, but of course, your journey is your own. Some examples of CSS Frameworks are:

  • Bootstrap: Build fast, responsive sites with Bootstrap
  • TailwindCSS: Rapidly build modern websites without ever leaving your HTML.
  • Bulma: The modern CSS framework that just works.
